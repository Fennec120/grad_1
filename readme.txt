# grad_1
intro_graduation
сберавтоподписка задачи
1) добиться roc-auc 0.65+
2) собрать сервис, который будет принимать определенные колонки на вход и отдавать значение 1  или 0

Проект разбит на 4 файла в порядке запуска

1) funcs - функции, к которым будет обращаться пайп
2) df_prep - подготовка урезанного датасета - обучать модель на 15 000 000 строк даже с увеличенной оперативкой показалось самоубийством
3) pipeline - обучение модели на урезанном датасете и соханнеие в pkl
4) main - сервис, описание моделей приема запросов, ответов на запросы

Запуск предполагается с помощью терминала и команды uvicorn main:app --reload
отправка запросов через постман на localhost:8000, тело запроса берется из 10и json'ов по адресу /resources
